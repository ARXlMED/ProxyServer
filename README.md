# ProxyServer
## What is this application? / Что из себя представляет данное приложение?
Данное приложение это прокси-сервер для http, который логирует все приходящие на него http заголовки. Также есть возможность задать черный список с помощью blacklist.txt и тогда прокси сервер не будет пропускать соединение с сайтом.
## Install / Установка
1. Открыть папку App и запустить приложение
2. Указать желаемый IP адрес и порт на LoopBack интерфейсе
3. Задать в настройках браузера (например Firefox) тот же IP адрес и порт для HTTP что и в приложении
## How to block connection? / Как заблокировать подлючение?
1. Открыть файл blacklist.txt
2. Указать домен (например bru.by)
3. Заблокирована не только главная страница, но и все остальные страницы сайта
